JavaScript

JavaScript — мультипарадигменный язык программирования. Поддерживает объектно-
ориентированный, императивный и функциональный стили. Является реализацией языка
ECMAScript. JavaScript обычно используется как встраиваемый язык для программного доступа
к объектам приложений. Наиболее широкое применение находит в браузерах как язык
сценариев для придания интерактивности веб-страницам.
Основные архитектурные черты: динамическая типизация, слабая типизация, автоматическое
управление памятью, прототипное программирование.
