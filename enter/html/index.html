HTML

HTML (от англ. HyperText Markup Language — «язык гипертекстовой разметки»).
Стандартизированный язык разметки документов во Всемирной паутине. Большинство
веб-страниц содержат описание разметки на языке HTML (или XHTML). Язык HTML
интерпретируется браузерами; полученный в результате интерпретации форматированный
текст отображается на экране монитора компьютера или мобильного устройства.

Текстовые документы, содержащие разметку на языке HTML (такие документы традиционно имеют расширение .html или .htm), обрабатываются специальными приложениями, которые отображают документ в его форматированном виде.

Структура HTML-документа

В HTML языке есть теги, а у тегов есть свои атрибуты.

В данном примере div- это тег, class- имя атрибута, а head-значение атрибута.
Существует огромное количество тегов и атрибутов.
